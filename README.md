# Garmin Fenix 7 Breathing App

Приложение для дыхательных упражнений на Garmin Fenix 7. Использует SDK Connect IQ 8.4.1.

## Цель

Приложение помогает пользователю:
- Сфокусироваться на дыхании
- Расслабиться через визуальные и вибрационные сигналы
- Отслеживать дыхательные сессии и прогресс

## Особенности

- Настраиваемая длительность дыхательных сессий
- Визуальные подсказки для вдоха и выдоха
- Вибрация для сигналов дыхания
- Поддержка разных режимов дыхания (например, 4-7-8, равномерное дыхание)

## Требования

- Garmin Fenix 7 или совместимые устройства
- Connect IQ SDK 8.4.1
- Eclipse / VS Code с поддержкой Monkey C
- Git для контроля версий
